# Desktop Assistant
Desktop Assistant written in python that takes voice input and performs tasks given by the user.
 
### Requirements
- OS: Windows
- Python 3.x
- IDE can be of your choice..e.g.,PyCharm, Python IDLE, VS Code, Spyder,etc.
